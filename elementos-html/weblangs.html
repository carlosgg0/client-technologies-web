El lenguaje de marcado HTML

Orígenes

A comienzos de la década de 1990, Tim Berners-Lee propuso un sistema para compartir información entre investigadores a través de la web. Para estructurar los documentos creó HTML, un lenguaje simple basado en etiquetas. Según el propio Tim Berners-Lee : La idea original de la web era que debía ser un espacio colaborativo donde poder comunicarse a través del intercambio de información

Primeras versiones

Las primeras especificaciones de HTML eran reducidas, pero suficientes para incluir títulos, párrafos, enlaces y listas. Gracias a esa simplicidad, HTML se adoptó con rapidez en universidades y centros de investigación.

Estandarización

Con el crecimiento de Internet, el W3C comenzó a publicar recomendaciones para mantener la compatibilidad entre navegadores. Este proceso permitió que HTML evolucionara de forma ordenada y que los desarrolladores tuvieran reglas comunes.

HTML en la actualidad

Hoy, HTML sigue siendo la base de la web moderna. Combinado con CSS y JavaScript, permite crear sitios accesibles, interactivos y compatibles con múltiples dispositivos, manteniendo la idea original impulsada por Tim Berners-Lee.

Características Principales de HTML

Versiones de HTML en orden cronológico

  1. HTML 2.0 (1995)
  2. HTML 3.2 (1997)
  3. HTML 4.01 (1999)
  4. XHTML 1.0 (2000)
  5. HTML5 (2014)

Ventajas de HTML5

  • Mayor compatibilidad con dispositivos móviles.
  • Etiquetas semánticas que mejoran la estructura del contenido.
  • Soporte nativo para audio y video sin complementos externos.
  • Mejor integración con APIs modernas del navegador.

Elementos de HTML por categorías

  • Estructura

    <html>
    Elemento raíz del documento HTML.
    <head>
    Contiene metadatos del documento.
    <body>
    Contiene el contenido visible del documento.

  • Texto

    <h1> a <h6>
    Encabezados de diferentes niveles.
    <p>
    Elemento para párrafos de texto.
    <strong>
    Elemento para texto en negrita.

  • Multimedia

    <img>
    Elemento para insertar imágenes.
    <audio>
    Elemento para reproducir audio.
    <video>
    Elemento para reproducir video.

  • Listas y navegación

    <ul> y <ol>
    Elementos para listas no ordenadas y ordenadas.
    <li>
    Elemento para elementos de lista.
    <a>
    Elemento para enlaces hipertexto.
